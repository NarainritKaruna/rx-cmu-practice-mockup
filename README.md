# Rx CMU Practice Mockup

Local static mockup for an Rx CMU-style OPD workflow.

This repository uses synthetic clinical content for practice and demonstration. Identifying fields such as names, HN/VN/TXN, PID, phone numbers, addresses, dates of birth, and staff identifiers are masked or replaced to avoid PDPA-sensitive content.

Open `index.html` directly in a browser, or serve the folder with a local static server.
